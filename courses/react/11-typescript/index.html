TypeScript

TypeScript is JavaScript with syntax for types

  • Статична типізація
  • Підтримка новітніх функцій JavaScript
  • Підтримка IDE

Астановлення

npm install typescript --save-dev

Ініціалізація

tsc --init - створює файлік tsconfig.json з налаштуваннями компілятора

Налаштування

  • noImplicitAny - помилка, якщо тип не вказаний або використано any
  • target - вказуємо, в яку версію ECMAScript компілювати
  • removeComments - видаляє коментарі
В package.json створюємо скрипт для запуска компілятора
            "scripts": {
              "compile": "tsc ./index.ts",
              "watch": "tsc ./index.ts --watch"
            },

Типи

  • Boolean: логічне значення true або false
  • Number: числове значення
  • String: рядок
  • Array: массив
  • Tuple: кортедж
  • Enum: перелічення
  • Any: довільний тип
  • Null і undefined: відповідяють значенням null і undefined в javascript
  • Void: відсутність конкретного значення, використовується в основному як тип, що повертається функціями.

Функції

            function add(a: number, b: number): number {
              return a + b;
            }
            let result1 = add(1, 2);
        
            function getFullName(firstName: string, lastName?: string) {
              if (lastName)
                return firstName + " " + lastName;
              else
                return firstName;
            }
