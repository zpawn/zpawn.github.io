Action Creators

Action Creators - ф-ция которая создает объект action.

            // userActions.js
            const userLogin = (login, password) => {
              return { type: 'USER_LOGIN', login, password }
            }
             
            // SomeComponent.js
            dispatch(userLogin('armen', '123'));

Middleware

Middleware

Middleware - это ф-ция, которая модифицирует ф-цию dispatch. И если у нас несколько middleware, то они будут вызываться по цепочке. 

            const logMiddleware = (store) => (next) => (action) => {
              console.log('[LOG_MIDDLEWARE]');
              return next(action);
            };

Redux Thunk

Thunk - позволяет передавать в store ф-цию, как action.

Устанавливаем: npm install redux-thunk
Подключаем:
            // store/index.js
            import thunk from 'redux-thunk'
             
            const store = createStore(reducers, applyMiddleware(thunk));
