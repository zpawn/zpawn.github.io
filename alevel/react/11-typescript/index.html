TypeScript

TypeScript is JavaScript with syntax for types

  • Статическая типизация
  • Поддержка новейших функций JavaScript
  • Поддержка IDE

Установка

npm install typescript --save-dev

Инициализация

tsc --init - создаст файлик tsconfig.json с настройками компилятора

Настройки

  • noImplicitAny - ошибка если тип не указан или any
  • target - указывает в какую версию ECMAScript компилировать
  • removeComments - даляет комментарии
В package.json создаем скрипт для запуска компилятора
            "scripts": {
              "compile": "tsc ./index.ts",
              "watch": "tsc ./index.ts --watch"
            },

Типы

  • Boolean: логическое значение true или false
  • Number: числовое значение
  • String: строки
  • Array: массивы
  • Tuple: кортежи
  • Enum: перечисления
  • Any: произвольный тип
  • Null и undefined: соответствуют значениям null и undefined в javascript
  • Void: отсутствие конкретного значения, используется в основном в качестве возвращаемого типа функций

            const a: number = 1;
            const b: string = 2;
            const c: boolean = false;
            const arr: number[] = [1,2,3,4];

            let userInfo: [string, number];

Функции

            function add(a: number, b: number): number {
              return a + b;
            }
            let result1 = add(1, 2);
        
            function getFullName(firstName: string, lastName?: string) {
              if (lastName)
                return firstName + " " + lastName;
              else
                return firstName;
            }

Классы

            class User {
              id: number;
              name: string;
              getInfo(): string {
                return "id:" + this.id + " name:" + this.name;
              }
            }
