Списки
---
Маркированный список
```html
<ul>
  <li>Первый пункт</li>
  <li>Второй пункт</li>
  <li>Третий пункт</li>
</ul>
```

Нумерованный список
```html
<ol>
  <li>Первый пункт</li>
  <li>Второй пункт</li>
  <li>Третий пункт</li>
</ol>
```

Вложенные списки
```html
<ul>
    <li>Первый пункт</li>
    <li>Второй пункт</li>
    <li>
        Треий пункт
        <ol>
          <li>Первый подпункт</li>
          <li>Второй подпункт</li>
          <li>Третий подпункт</li>
        </ol>
    </li>
</ul>
```

Таблицы
---

`<table></table>` - контейнер для всей таблицы

`<tr></tr>` - строки

`<td></td>` - ячейки

Простая таблица
```html
<table>
    <tr>
        <td>1</td>
    </tr>
</table>
```

Объединение ячеек по горизонтали (колонок)
```html
<table>
    <tr>
        <td>1</td>
        <td>2</td>
    </tr>
    <tr>
        <td colspan="2">3</td>
    </tr>
</table>
```

Объединение ячеек по вертикали (строк)
```html
<table>
    <tr>
        <td rowspan="2">1</td>
        <td>2</td>
    </tr>
    <tr>
        <td>3</td>
    </tr>
</table>
```
